Q:How does javaScript work?

Ans:JavaScript is a synchronous single-threated programming language.Everything in JavaScript happens inside an Execution Context.An execution context to be a big box or a container in which whole javaScript code is executed.Execution context has two componants in it.The first components is also known as the memory components and here all the variable and function are stored.The second of the execution context is the code component.This is the place where code is executed one line at a time.


Q:What is the difference between local storage & session storage?

Ans:Local storage and session storage both are used to stored data only client site.The differences between local storage and session storage are given bellow: (1)Local storage capability to stored data 10MB but the session storage is only 5MB. (2)Session storage has a expiry but the local storage doesn't have expire. (3)Local storage has the heigher memory capacity but session does not have. (4)Local storage save any kind of data which we need to store even when browser is closed,but session storage's data will be lost when browser is closed.
